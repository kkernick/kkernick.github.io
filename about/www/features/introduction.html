The Features section contains information on:

Main Panel
A list of features for customizing your heatmap that are available on the main sidebar.

Advanced Panel
A list of features for customizing your heatmap that are available in the advanced options panel. These features can be accessed by clicking the "Show Advanced Options" button on the main sidebar.

The Features section contains an overview of all the options for customizing your heatmap. Please select a heatmap type from the sidebar to view its specific features.
